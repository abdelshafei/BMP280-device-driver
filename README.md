# BMP280 device driver 

## BMP280 Sensor Connection

| BMP280 Pin | Raspberry Pi 4 Connection  |
|------------|----------------------------|
| VCC        | 3.3V (pin 1)               |
| GND        | Ground (pin 6)             |
| SCL        | GPIO 3 (pin 5)             |
| SDA        | GPIO 2 (pin 3)             |
